# OpenGenome De Novo Assembly Output

| Directory | Contents |
| --- | --- |
| `reads/` | Staged long-read FASTQ input. |
| `qc/` | `seqkit stats` read summaries. |
| `assembly/` | Primary FASTA/GFA, assembler logs, gfastats output, circularity review TSV, read-density TSV/SVG, and assembly graph SVG preview. |
| `report/` | `denovo_assembly_report.html`, `denovo_assembly_summary.tsv`, `denovo_assembly_manifest.json`, with embedded read-density, assembly-graph, and circularity previews plus links to the underlying review artifacts. |
| `pipeline_info/` | Nextflow timeline, report, trace, and DAG files. |

The report contract is assembler-neutral: hifiasm, Flye, and Verkko branches all publish the same summary fields for the TUI. The circularity table is a conservative review aid based on terminal overlap and GFA links; it is not an automatic biological interpretation. The read-density plot is generated by aligning staged long reads back to the assembled contigs and binning mean depth across contig windows.
